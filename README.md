# Portfolio Website

A modern, responsive personal portfolio website showcasing professional experience, education, projects, and technical expertise.
Link - https://hemanath17.github.io/hemanath_portfolio/

## Overview

This portfolio website is built using HTML, CSS, and vanilla JavaScript, featuring a clean and professional design. It includes sections for:

- **About**: Personal introduction and background
- **Education**: Academic qualifications with timeline visualization
- **Experience**: Professional work history with company logos and links
- **Projects**: Portfolio of completed projects
- **Expertise**: Technical skills organized by categories
- **Contact**: Contact information and form

## Features

- Responsive design that works on all devices
- Timeline-based layout for education and experience sections
- Interactive project gallery
- Smooth scrolling navigation
- Animated typing effect in hero section
- Modern card-based design for expertise section

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap (Grid system)
- AOS (Animate On Scroll)
- Typed.js (Typing animation)
- Various icon libraries (Boxicons, Icofont)

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. For production deployment, upload all files to a web server (e.g. GitHub Pages)

### Contact form (GitHub Pages / static hosting)

The contact form uses [Web3Forms](https://web3forms.com) so it works without PHP.

Submissions are sent to the email linked to your Web3Forms access key in the dashboard (currently **findhemanath@gmail.com**). To change the recipient or domain restrictions, edit settings at [web3forms.com](https://web3forms.com).

The legacy `forms/contact.php` file is not used when the form posts to Web3Forms.
